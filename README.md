# 📊 Mutual Fund Analytics Capstone Project

## 📌 Overview
This project focuses on analyzing mutual fund datasets using Python and real-time NAV data from APIs. It demonstrates data engineering, validation, and analytics workflows.

---

## 🎯 Objective
To build a structured data pipeline for mutual fund analysis using multiple datasets and live market data.

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Requests

---

## 📁 Project Structure
- data/ → Raw datasets and NAV files
- src/ → Python scripts for processing
- reports/ → Internship documentation
- notebooks/ → Data analysis work
- sql/ → SQL queries (future use)
- dashboard/ → Visualization (future use)
